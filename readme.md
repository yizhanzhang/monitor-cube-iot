# MonitorCube

MonitorCube is a simple IOT device, base on esp 8266.
It was created by yizhanzhang just for fun.

## Pin Config

|       |        |
| ----  | ----   |
| SCK   | GPIO14 |
| MOSI  | GPIO13 |
| RES   | GPIO2  |
| DC    | GPIO0  |
| LCDBL | GPIO5  |
